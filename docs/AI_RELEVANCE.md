# AI Relevance Scoring

## Overview

AI relevance scoring filters news items to surface AI-related content and reduce noise. The scoring system is configurable per source type.

## Configuration

| Source Type | Default Score |
|---|---|
| Official AI sources | 0.35 |
| AI Hot / AIBase | 0.45 |
| Follow Builders | 0.25 |
| OPML RSS | 0.15 |
| X API | 0.15 |
